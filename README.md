# Modern Gym Landing Page & Client Portal

A high-performance, responsive web application built with React, TypeScript, and Vite, designed specifically for modern fitness centers. This platform seamlessly integrates conversion-focused marketing sections with essential client-side utility features.

## Core Features

*   **Dynamic BMI Calculator:** Provides instantaneous body mass index calculations coupled with real-time health category classification.
*   **Tiered Membership Architecture:** Structured pricing grids explicitly detailing tier features, premium benefits, and call-to-actions.
*   **Interactive Workout Log:** A state-of-the-art client tracking portal allowing users to log daily exercise routines, record sets/reps, and view history.
*   **Performance Optimization:** Developed using lightweight components and Vite's ultra-fast bundling to ensure sub-second load times on mobile and desktop devices.

## Tech Stack

*   **Frontend Framework:** React (TypeScript)
*   **Build Tool:** Vite
*   **Styling:** Modern CSS / Tailwind CSS
*   **Deployment:** Vercel
